# Tableau Customization Map (Grouping)

An interactive US map visualization that groups states by custom regions 
and displays aggregated values per state, built in Tableau.

## 🛠 Tools Used
- Tableau Desktop / Tableau Public
- State-level grouping and custom color-coded regions

## 📌 Key Features
- Custom grouping of US states into 4 regions (color-coded)
- Value labels displayed per state
- Interactive filtering and zoom
